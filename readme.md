<h2>🚀 Getting Started</h2>
<p>

</p>

<pre><code>git clone https://github.com/FabihaPritha/Brainwave_Matrix_Intern.git
cd quiz_app  # or cd news_reader_app
flutter pub get
flutter run
</code></pre>
